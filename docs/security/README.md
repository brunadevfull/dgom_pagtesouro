# Relatórios de Segurança

Este diretório é onde eu deixei organizado o material da análise estática de segurança do sistema PagTesouro DGOM.

## Conteúdo
- `analysis.md`: relatório principal com escopo, visão geral e achados.
- `evidence.md`: evidências e trechos de código citados (com caminhos e linhas).
- `recommendations.md`: plano de mitigação priorizado (quick wins, melhorias estruturais e dívida técnica).

## Escopo
Análise baseada exclusivamente no código disponível neste repositório, sem execução do sistema.
